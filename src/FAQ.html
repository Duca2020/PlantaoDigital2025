FAQ

Esta página é destinada a explicar as principais funcionalidades do sistema.

Sistema Plantão Digital

O Sistema Plantão Digital foi pensado como forma de digitalizar a maioria dos serviços do Plantão Funerário, oferecendo mais conforto e comodidade para o cidadão e facilidade de antendimento para os plantonistas.

Tela Incial

A tela inicial do Plantão Digital apresenta as principais funcionalidades do sistema, assim como algumas dicas de como tirar melhor proveito dos serviços oferecidos

O Menu Lateral

O Menu Lateral apresenta as principais ações e ferramentas dentro do sistema. Os serviços oferecidos variam conforme o grau de acesso do usuário, serviços como cadastro de novos processos só podem ser acessados por usuários internos, enquanto a procura de processos é uma ação disponibilizada para público amplo.

Cadastro

As ações aqui listadas são consideradas sob o ponto de vista geral, tanto ações de usuário interno quanto ações de usuário externo.

Novo Cadastro

Permite o cadastro de um novo processo, as informações de sessão são mantidas.
